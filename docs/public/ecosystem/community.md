# 社区

欢迎来到 A2UI 社区。我们正在一起构建智能体驱动界面的未来。

## 参与方式

A2UI 是一个采用 Apache 2.0 许可证的开源项目。我们欢迎开发者、研究者，以及任何对推进智能体用户界面感兴趣的人一起参与。

## 社区展示

如果你用 A2UI 做了什么，欢迎展示出来。

分享作品的最佳方式，是在 **[GitHub Discussions](https://github.com/a2ui-project/a2ui/discussions)** 发帖，附上简短说明、代码链接，以及一个不超过 4 分钟的演示视频。我们会把值得关注的项目收录到文档和发布说明中。

**由社区构建的渲染器** 会列在 [生态渲染器](renderers.md) 页面中，你可以在那里查看社区实现，以及如何把自己的实现加进去。

[仓库中的官方示例](../../../samples/README.md) 很适合作为参考，帮助你理解一个高质量的 A2UI 项目应该是什么样子。

## 项目合作伙伴

A2UI 与 Google Opal、Flutter（GenUI SDK）、Gemini Enterprise、AG-UI / CopilotKit 以及 A2A 团队共同协作开发。更多详情与案例可参见 [A2UI in the World](a2ui-in-the-world.md)。

如果你对 A2UI 做出了重要贡献，也欢迎把自己的名字提交到这个列表中。

## 行为准则

我们致力于为所有人提供一个友好、包容的环境。所有参与者都应当：

- 保持尊重与体谅
- 欢迎新人并帮助他们开始参与
- 关注什么对社区最有利
- 对他人保持同理心

我们不会容忍不可接受的行为。如有顾虑，请向项目维护者反馈。

## 致谢与认可

我们感谢所有贡献。贡献者会在以下地方获得认可：

- GitHub 贡献图谱
- 发布说明
- 项目致谢

贡献显著的参与者，可能会被邀请加入项目维护团队。

## 保持关注

- **Watch GitHub 仓库** 获取通知
- **Star 仓库** 以便收藏并表达支持
- **关注 Releases** 以便及时获知新版本

## 贡献方式

**[github.com/a2ui-project/a2ui](https://github.com/a2ui-project/a2ui)**

- **报告问题**：发现 bug？[提交 issue](https://github.com/a2ui-project/a2ui/issues)
- **构建渲染器**：计划中的框架可查看 [路线图](../roadmap.md)
