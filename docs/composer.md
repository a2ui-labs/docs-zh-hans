# A2UI Composer

使用 **CopilotKit A2UI Widget Builder** 以交互方式构建 A2UI 小部件。

[![A2UI Composer](assets/A2UI-widget-builder.png)](https://go.copilotkit.ai/A2UI-widget-builder)

**[启动 Widget Builder →](https://go.copilotkit.ai/A2UI-widget-builder)**

## 它能做什么

- 以可视化方式试验 A2UI 组件
- 通过描述你的需求来生成 A2UI JSON
- 查看实时预览
- 复制 JSON 并用于你的智能体

由 [CopilotKit](https://www.copilotkit.ai/) 团队打造。
